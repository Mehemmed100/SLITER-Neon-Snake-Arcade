# 🐍 SLITER — Neon Snake Game

A fast-paced, retro-futuristic snake game inspired by classic arcade mechanics. Eat, grow, survive — and try not to crash.

## 🎮 How to Play

* **↑ ↓ ← →** or **W A S D** — Move the snake
* Eat **green food** → Grow and gain points
* Catch the **pink star** → Get **50× score boost** (temporary)
* Don’t hit the **walls** or **yourself** → Game over
* Press **ESC** → Pause the game

## 🚀 Features

* ⚡ Dynamic speed system (increases every 50 points, Level 1 → 5)
* ✨ Particle effects when eating food
* 💾 High score saved using localStorage
* 📱 Mobile touch support
* 🌈 Neon / retro-futuristic design

## 🧠 Gameplay Mechanics

The more you grow, the harder it becomes to control your snake. Speed increases over time, making quick reflexes and smart movement essential.

Special pink stars appear randomly — grab them fast to boost your score massively, but don’t risk crashing while chasing them.
